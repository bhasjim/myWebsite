My Projects

Exposure

This is a map-based website that displays a heat map of popular photo op locations nearby. Able to filter by tags and search by locations. Leverages Flickr’s API. Built with HTML/CSS, Node/Express, JS, and Google Maps. Check it out here!

Tweeter

A clone of the mobile Twitter app. I made this to learn how to use Swift and XCode, and because I wanted to learn how to develop mobile apps. I utilized Twitter's API to support tweets, retweets, favorites, and OAuth login. Check out the gif!

Flicks

A simple iOS app that views movie titles and posters from a movie database. Like the Tweeter app, I created this app to learn about mobile app development, and also to learn how to use public APIs. Check out the gif!

WorldCare Technologies Website

A website idea for Worldcare Technologies, a UCSD club turned benefit corporation dedicated to building an affordable and reliable HIV test for use in the developing world. Built in NodeJS/HTML/CSS.

Spots

One of my first web pages that I created. Spots is a webapp that allows users to log their travel locations, and also outputs some analytics on their adventures such as how often a user travels. Built in NodeJS/Jade/CSS. Check it out here!

Tipper

My very first iOS app. Just a simple tip calculator. Check out the gif!
